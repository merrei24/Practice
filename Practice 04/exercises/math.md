# Python Math

## Degree to radian
This converts degrees to radians using math module.

```python
import math

degree = 15
print(math.radians(degree))
```

## Area of trapezoid
This calculates trapezoid area using formula.

```python
h = 5
a = 5
b = 6

print((a + b) * h / 2)
```

## Area of regular polygon
This calculates area of regular polygon.

```python
import math

n = 4
a = 2

area = (n * a * a) / (4 * math.tan(math.pi / n))
print(area)
```

## Area of parallelogram
This calculates area of parallelogram.

```python
base = 5
height = 6

print(base * height)
```
